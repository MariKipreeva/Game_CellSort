УСТАНОВКА ШРИФТОВ:

1. Откройте папку "Fonts".
2. Откройте zip-архив "Consolar", далее нажните на файл с 
индентичным названием, нажмите кнопку "Установить" в левом верхнем углу.
3. Готово шрифт установлен.
4. Повторите шаг 2 остальными файлами папки "Fonts".
Готово!

-----------------------------------------------------------

ЗАПУСК ИГРЫ:

1. Откройте папу Game.
2. Запустите приложение "Flask.exe".
Готово!

-----------------------------------------------------------

НАЧАТЬ ИГРУ ЗАНОВО C 1 УРОВНЯ:

1. Зайдите в папку Saves.
2. В файле "LevelSave.txt" удалите все содержимое и оставьте файл пустым.
3. В файле "Level.txt" удалите все содержимое и напишите 1.
4. В файле "Score.txt" удалите все содержимое и напишите 0.
5. Перезапустите игру.
Готово!
